💡 Comment fonctionne la synchronisation admin → site

Ce site utilise le stockage local du navigateur (localStorage). Cela signifie que vos modifications sont enregistrées uniquement sur cet appareil et ce navigateur. Si un visiteur ouvre le site sur son téléphone ou un autre ordinateur, il ne verra pas vos photos et configurations personnalisées.

Pour que vos modifications soient visibles par tout le monde sur internet, il faut déployer le site sur un hébergement web :

✅ Solution simple et gratuite : GitHub Pages + Netlify
1. Créez un compte sur github.com et uploadez index.html + admin.html
2. Connectez à netlify.com → votre site sera en ligne en 2 minutes
3. Pour les photos et données : utilisez Airtable (gratuit) comme base de données — je peux vous aider à l'intégrer
🚀 Solution professionnelle : Supabase + déploiement automatique
Chaque modification admin se synchronise en temps réel sur tous les appareils. Nécessite quelques heures de développement supplémentaire — à prévoir pour la mise en ligne définitive.

En attendant le déploiement, utilisez ce dashboard pour préparer tout le contenu, puis exportez les données (bouton ci-dessus) pour les transférer vers le serveur final.
